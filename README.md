# glue-athena-cdk-example

A simple walkthrough to use AWS Glue, s3 and Athena.

Throughout the  `infra` folder has a Python/CDK project to create everything that you need to run this example.

## How To

### Requisites

* An AWS Account
* Python 3.8+
